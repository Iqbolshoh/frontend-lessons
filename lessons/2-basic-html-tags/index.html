Lesson 2: Basic HTML Tags

Learn the fundamental HTML tags and how to use them to build your webpage.

Most Common HTML Tags

<h1> - <h6>

Heading tags for titles and subtitles

<p>

Paragraph text

<a>

Hyperlinks to other pages

<img>

Embed images

<ul>, <ol>, <li>

Lists (unordered and ordered)

<div>

Content division container

HTML Document Structure


<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>
