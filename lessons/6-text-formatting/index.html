Lesson 6: Text Formatting Tags

Learn how to format text using bold, italic, underline, and other tags in HTML.

Text Formatting in HTML

In HTML, there are specific tags used to format text such as <b>, <i>, <u>, <strong>, <em>, <s>, <mark>, <small>, <del>, <sub>, and <sup>.

Bold Text: <b>

The <b> tag is used to make text bold.

<b>This text is bold</b>

Italic Text: <i>

The <i> tag is used to italicize text.

<i>This text is italic</i>

Underlined Text: <u>

The <u> tag is used to underline text.

<u>This text is underlined</u>

Strong Text: <strong>

The <strong> tag is used to make text strongly emphasized, usually appearing in bold. 

<strong>This text is strong</strong>

Emphasized Text: <em>

The <em> tag is used to emphasize text, usually appearing in italic.

<em>This text is emphasized</em>

Strikethrough Text: <s>

The <s> tag is used to represent text that is no longer correct or relevant, often shown with a line through it.

<s>This text is strikethrough</s>

Marked Text: <mark>

The <mark> tag is used to highlight text, usually with a yellow background.

<mark>This text is highlighted</mark>

Smaller Text: <small>

The <small> tag is used to display text in a smaller size.

<small>This text is smaller</small>

Deleted Text: <del>

The <del> tag is used to show text that has been deleted or is no longer relevant.

<del>This text is deleted</del>

Subscript Text: <sub>

The <sub> tag is used for subscript text, which is usually smaller and positioned below the regular line of text.

<sub>This is subscript</sub>

Superscript Text: <sup>

The <sup> tag is used for superscript text, which is usually smaller and positioned above the regular line of text.

<sup>This is superscript</sup>

Note: You can combine these tags to apply multiple formatting styles to text.
