Lesson 11: Linking External Files

Learn how to link external resources like CSS, JavaScript, Font Awesome, and Google Fonts to your webpage.

Why Use External Files?

External files help keep your HTML clean, reusable, and easier to maintain. You can style multiple pages with one CSS file or run scripts with one JS file.

1. Linking CSS

Use the <link> tag inside the <head> to include a CSS file:

<link rel="stylesheet" href="styles.css">

2. Linking JavaScript

Use the <script> tag before the closing </body> tag:

<script src="scripts.js"></script>

3. Using Font Awesome

Font Awesome gives you access to thousands of icons:

<script src="https://kit.fontawesome.com/your-kit-id.js"></script>
<i class="fas fa-user"></i>

4. Adding Google Fonts

You can add beautiful fonts by copying the link from Google Fonts:

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

<style>
    body {
        font-family: 'Roboto', sans-serif;
    }
</style>

Tip: Place CSS links in the <head> and JavaScript links before the closing </body> tag for better performance.
