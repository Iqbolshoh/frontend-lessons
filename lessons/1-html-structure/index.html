Lesson 1: HTML Structure

Learn the basic structure of HTML and how to create your web page.

Basic Tags in HTML

HTML uses various tags to build a web page. Here are the basic ones:

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document.
  • <html> - The root element of an HTML page.
  • <head> - Contains meta information about the page.
  • <body> - Contains the visible page content.
  • <h1> to <h6> - Headings from largest (h1) to smallest (h6). (Heading)
  • <p> (Paragraph) - Paragraph tag to write text.
  • <a> - (Anchor) Anchor tag to create links.
  • <img> - (Image) Image tag to display pictures.
  • <button> - (Button) Creates a clickable button.
  • <iframe> - (Iframe) Embeds another webpage inside your page.
  • <br> - (Line Break) Line break to create space between elements.
  • <hr> - (Horizontal Rule) Horizontal rule to separate content visually.

Example of HTML Code

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first web page.</p>
    
    <br>
    
    <hr>
    <a href="https://iqbolshoh.uz">Click here</a> to visit the website.
    
    <img src="example.jpg" alt="Example Image" width="300">
    
    <button>Click Me!</button>
    
    <iframe src="https://iqbolshoh.uz" width="400" height="300"></iframe>
    
  </body>
</html>
