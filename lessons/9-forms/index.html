Lesson 9: Basic Form Elements

Learn how to create forms in HTML and understand the basic elements required for building forms.

HTML Forms

Forms are used to collect user input. They are essential for interactive websites.

Basic Structure

  • <form> – Defines the form.
  • <input> – Defines an input field.
  • <label> – Describes an input field.
  • <textarea> – Multiline text input.
  • <select> – Drop-down menu.
  • <button> – Button to submit or reset.
<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br><br>

    <label for="message">Message:</label><br>
    <textarea id="message" name="message"></textarea><br><br>

    <label for="gender">Gender:</label>
    <select id="gender" name="gender">
        <option value="male">Male</option>
        <option value="female">Female</option>
    </select><br><br>

    <button type="submit">Submit</button>
</form>
