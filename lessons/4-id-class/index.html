Lesson 4: ID and Class Usage

Learn the differences between ID and class and how to apply them in styling HTML elements.

What are ID and Class in HTML?

ID and class are used to apply styles to specific HTML elements or groups of elements. The ID is unique to a single element, while classes can be applied to multiple elements.

Difference between ID and Class:

  • ID – Identifies a unique element on the page. It should be used only once on a page.
  • Class – Can be applied to multiple elements. It is used for grouping elements with similar styles.

Example Usage of ID and Class:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ID and Class Usage</title>
</head>
<body>

<div id="uniqueElement" style="background-color: lightblue; padding: 20px; border-radius: 10px;">
    This is an element with a unique ID.
</div>

<div class="commonClass" style="background-color: lightgreen; padding: 20px; border-radius: 10px;">
    This is an element with a class applied.
</div>

<div class="commonClass" style="background-color: lightgreen; padding: 20px; border-radius: 10px;">
    This is another element with the same class.
</div>

</body>
</html>

Note: The ID attribute must be unique on the page, while the class attribute can be reused on multiple elements.
