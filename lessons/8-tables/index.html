Lesson 8: Creating Tables

Learn how to create tables in HTML and how to use them to organize data.

Tables in HTML

In HTML, you can create tables using the <table> tag. A table is made up of rows and columns.

Basic Table Structure

The basic structure of a table includes:

  • <table> – Defines the table.
  • <tr> – Defines a table row.
  • <th> – Defines a table header cell.
  • <td> – Defines a table data cell.
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
    <tr>
        <td>Data 3</td>
        <td>Data 4</td>
    </tr>
</table>

Table with Borders

You can add borders to your table using the border attribute or through CSS:

<table border="1">
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Table with CSS Styling

To style tables more effectively, you can use CSS to add styles like borders, padding, and text alignment:

<style>
    table {
        width: 100%;
        border-collapse: collapse;
    }
    th, td {
        border: 1px solid black;
        padding: 8px;
        text-align: left;
    }
</style>
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Table with Colspan and Rowspan

You can also merge cells using colspan (to span across columns) and rowspan (to span across rows):

<table>
    <tr>
        <th colspan="2">Merged Header</th>
    </tr>
    <tr>
        <td rowspan="2">Merged Cell</td>
        <td>Data 1</td>
    </tr>
    <tr>
        <td>Data 2</td>
    </tr>
</table>

Note: HTML tables are useful for displaying data in rows and columns. You can also use CSS to further style and organize your tables.
