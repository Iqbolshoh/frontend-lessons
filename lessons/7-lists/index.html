Lesson 7: Lists (Ordered and Unordered)

Learn how to create ordered and unordered lists in HTML.

Lists in HTML

HTML provides two types of lists: ordered lists and unordered lists.

Ordered List: <ol>

The <ol> tag is used to create ordered (numbered) lists. Each item in the list is wrapped in a <li> tag.

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Unordered List: <ul>

The <ul> tag is used to create unordered (bulleted) lists. Each item is wrapped in a <li> tag.

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Ordered List with Custom Types

You can customize the ordered list using the type attribute:

  • Type "1": Numbers (1, 2, 3, etc.)
  • Type "a": Lowercase letters (a, b, c, etc.)
  • Type "A": Uppercase letters (A, B, C, etc.)
  • Type "i": Lowercase Roman numerals (i, ii, iii, etc.)
  • Type "I": Uppercase Roman numerals (I, II, III, etc.)
<ol type="a">
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Unordered List with Custom Types

You can also customize the unordered list with the type attribute:

  • Type "disc": Filled circular bullets (default)
  • Type "circle": Hollow circular bullets
  • Type "square": Square bullets
<ul type="square">
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Nested Lists

You can also create lists inside other lists, called nested lists. This can be done by placing an <ul> or <ol> list inside a <li> item of another list.

<ul>
    <li>First item</li>
    <li>
        Second item with a nested list:
        <ol>
            <li>Nested item 1</li>
            <li>Nested item 2</li>
        </ol>
    </li>
    <li>Third item</li>
</ul>

Note:

Ordered lists (<ol>) will display items with numbers or letters depending on the type specified. Unordered lists (<ul>) will display items with bullets by default.
