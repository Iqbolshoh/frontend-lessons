Lesson 3: HTML Attributes (Style)

Learn how to use HTML attributes, especially the style attribute to customize your elements.

What are HTML Attributes?

Attributes provide additional information about HTML elements and are specified in the opening tag.

Common HTML Attributes:

  • href – For links (used with <a> tags)
  • src – For image sources (<img> tag)
  • alt – Provides text description for images
  • style – Allows inline CSS directly in an element
  • width and height – Set element size
  • background-color – Defines the background color of an element
  • border-radius – Rounds the corners of an element
  • color – Specifies the text color
  • font-size – Specifies the size of the text
  • font-weight – Controls the thickness of the text (e.g., bold)
  • font-style – Defines italic, normal, or oblique text styles
  • text-align – Controls text alignment: left, center, or right
  • font-family – Defines the font type (e.g., Arial, Roboto, etc.)

Using the style Attribute in HTML Tags:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Attributes (Style)</title>
</head>
<body>

<p style="color: #444; font-size: 18px; font-family: 'Georgia'; background-color: #f0f0f0; padding: 10px;">
    This paragraph is styled with background, font, and color.
</p>

<a href="#" style="color: blue; text-decoration: none; font-weight: bold;">
    Visit Our Page
</a>

<img src="example.jpg" alt="Example Image" style="width: 300px; height: auto; border-radius: 15px; border: 2px solid #ccc;">

<div style="background-color: lightblue; padding: 20px; border-radius: 10px; width: 400px;">
    <p style="margin: 0;">This is a styled box with padding and border-radius.</p>
</div>

<button style="background-color: green; color: white; font-size: 18px; padding: 10px 20px; border: none; border-radius: 5px;">
    Click Me
</button>

</body>
</html>

Note: You can write multiple CSS properties inside the style attribute by separating them with a semicolon ;.
