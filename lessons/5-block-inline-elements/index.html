Lesson 5: Block and Inline Elements

Understand the differences between block and inline elements and how to use them effectively.

What are Block and Inline Elements?

HTML elements can be divided into two categories: block-level elements and inline elements. Block elements take up the full width of their parent container, while inline elements only take up as much width as necessary to contain their content.

Block Elements:

  • <div> – A container used for grouping elements together
  • <h1> to <h6> – Header tags used to define headings
  • <p> – Paragraph element used for text blocks
  • <ul>, <ol> – Lists (unordered and ordered)
  • <section> – Defines a section of content
  • <article> – Defines an independent piece of content
  • <footer> – Represents the footer of a page or section

Inline Elements:

  • <span> – A container for text or other inline elements
  • <a> – Defines a hyperlink
  • <img> – Embeds an image
  • <strong> – Defines strong emphasis (bold)
  • <em> – Defines emphasized text (italic)
  • <code> – Displays code snippets
  • <abbr> – Represents an abbreviation

Example Usage of Block and Inline Elements:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Block and Inline Elements</title>
  </head>
  <body>

  <div style="background-color: lightblue; padding: 20px; border-radius: 10px;">
      This is a block-level element.
  </div>

  <span style="color: red;">This is an inline element.</span>

  <p style="background-color: lightgreen; padding: 10px;">This is a paragraph element, which is block-level.</p>

  <a href="https://example.com" style="color: blue;">Click here</a>

  <footer style="background-color: gray; padding: 10px;">
      This is a block-level footer element.
  </footer>

  <article style="background-color: lightyellow; padding: 20px;">
      This is an article, a block-level element.
  </article>

  <span style="color: green;">This is another inline element.</span>

  <p> <abbr title="HyperText Markup Language">HTML</abbr> is the standard markup language for creating web pages. </p>  

  </body>
</html>

Note: Block elements take up the full width of the container, while inline elements only take up as much width as necessary.
