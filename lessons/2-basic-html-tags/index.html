Lesson 2: Basic HTML Tags

Learn the fundamental HTML tags and how to use them to build your webpage.

Important HTML Tags

HTML provides many tags to structure and design a webpage. Here are the most important ones:

  • <!DOCTYPE html> - Defines the document type and version (HTML5).
  • <html> - Root of an HTML document.
  • <head> - Meta information like title, links, etc.
  • <meta> - Defines metadata (charset, viewport, etc.).
  • <title> - Sets the title of the webpage (in browser tab).
  • <link> - Links external files like CSS.
  • <style> - Adds internal CSS styles.
  • <script> - Adds JavaScript code inside HTML.
  • <body> - Contains all visible content.
  • <h1> to <h6> - Heading tags (h1 largest, h6 smallest).
  • <p> - Paragraphs of text.
  • <br> - Line break.
  • <hr> - Horizontal line (divider).
  • <a> - Anchor/link to another page or site.
  • <img> - Displays images.
  • <button> - Clickable button element.
  • <div> - Division container to group content.
  • <span> - Inline container for text or elements.
  • <iframe> - Embeds another webpage inside the current page.

Example of Basic HTML Tags

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic HTML Tags Example</title>
    <link rel="stylesheet" href="styles.css">
    <style>
      body { 
        font-family: Arial, sans-serif; 
      }
    </style>
  </head>
  <body>

    <h1>Main Heading (h1)</h1>
    <h2>Subheading (h2)</h2>
    <h3>Smaller heading (h3)</h3>
    <h4>Even smaller heading (h4)</h4>
    <h5>Mini heading (h5)</h5>
    <h6>Tiny heading (h6)</h6>

    <p>This is a paragraph explaining basic HTML tags.</p>

    <a href="https://iqbolshoh.uz" target="_blank">Visit Iqbolshoh's Website</a>

    <br><br>

    <img src="https://via.placeholder.com/150" alt="Placeholder Image">

    <hr>

    <button onclick="alert('Button clicked!')">Click Me!</button>

    <div style="margin-top: 20px; padding: 10px; border: 1px solid #ccc;">
      <p>This is a <span style="color: red;">span</span> inside a div.</p>
    </div>

    <iframe src="https://example.com" width="300" height="200" title="Example iframe"></iframe>

    <script>
      console.log("Welcome to HTML lesson!");
    </script>

  </body>
</html>
